Collection List Page — Design Overview

The /collections page is rebuilt by replacing list-collections.json with stacked existing sections. No new Liquid files needed.

Page Structure

The template contains 5 sections in sequence, all fully configurable in the Shopify customizer:

1. Featured Collection — e.g. "The Oversized Bags"
Section type: featured-collection | Layout: grid, 4 cols desktop / 2 mobile | _section-header block for title + tagline
2. Promo Banner — "Yes, you need another one."
Section type: banner-grid | 3 cells: [text panel 2 cols] + [lifestyle image] + [lifestyle image]

What changes in the codebase

  • templates/list-collections.json — rewritten with the 5 sections above and sensible default settings. The main-list-collections section is removed.
  • No new Liquid files — all sections and blocks already exist in the theme.
  • Shopify customizer — merchant selects collections, uploads promo images, edits copy in the theme editor.

Key configuration for each Featured Collection section

  • Layout: Grid (not slideshow), 4 desktop cols / 2 mobile cols
  • Product limit: configurable (e.g. 4–8 per collection)
  • _section-header block: heading = collection name, text = playful tagline
  • button-group block: optional "View all" CTA linking to the collection page

Key configuration for each Banner Grid (promo) section

  • 3 _banner-grid-item blocks: columns 2+1+1 on desktop, stacked on mobile
  • Left cell: _grid-image-banner with no image — uses a color scheme as background, heading-group block for title + body, button-group for CTA
  • Center + Right cells: _grid-image-banner with lifestyle images, no text overlay
